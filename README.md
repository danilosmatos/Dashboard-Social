# Dashboard-Social
## Projeto de Modelagem de Dados

O projeto simula um processo real de Engenharia de Software: entender requisitos, modelar o sistema, criar um protótipo que consome uma API externa e defender a arquitetura.

### Tecnologias Usadas
>Python 3.10+
>
>JavaScript
>
>HTML & CSS

### Estrutura do Projeto

Baseado no modelo MVC (Model, View e Controller), junto do services que faz a comunicação com a API e o Main para rodar.

### Execução do Projeto

Com o projeto importado, rode o services.py primeiro e depois usando a extensão go live do vscode vá para o index.html e use ela.
